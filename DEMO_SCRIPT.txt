[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's walk through the architecture diagram to see how it works under the hood. 
Every query starts at the Security Checkpoint. This node performs heuristic checks for PII leakage, prompt injections, and role-based access control. 
If the query is clean, it is passed to our Strategic Advisor Agent. This orchestrator coordinates two specialized sub-agents: the Forecasting Agent, which calculates growth metrics, and the Risk Analysis Agent, which identifies anomalies. Both sub-agents query our datasets securely using a custom Model Context Protocol server.
Finally, the findings go through our Router Node. If the router detects high-priority anomalies or revenue drops, it flags the report as pending executive review. Otherwise, standard reports are automatically approved.
