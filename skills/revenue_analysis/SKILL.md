# Revenue Analysis Skill

**Goal**: Identify changes in monthly revenue, compare performance across regions, and assess product category shifting.

**Steps**:
1. Retrieve dataset content for the specified dataset (e.g. "sales").
2. Call the `run_analysis` capability tool with `analysis_type="revenue"`.
3. Extract percentage drop/growth month-over-month.
4. Identify which product categories or regions caused the drop.
5. Formulate findings clearly highlighting regional variances (e.g., East Region decline) and category drops (e.g., Product Category B).
